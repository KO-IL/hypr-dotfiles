# KO-IL Arch Hyprland 桌面模板系统
---

## 🧠 项目理念

> 环境即代码（Environment as Code）

本仓库包含：

- 桌面配置文件
- 官方仓库软件列表
- AUR 软件列表
- 一键自动部署脚本
- 可跨设备恢复的完整环境
---

## ⚙ 功能特性

- Hyprland 窗口管理器
- Waybar 状态栏
- Rofi 启动器
- Kitty 终端
- Git 管理 dotfiles
- 官方软件自动恢复
- AUR 自动安装
- SSH 安全部署
- 一键环境重建

---
